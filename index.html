Ultimate Pong

This is a 2D Pong game created using JavaScript with ChatGPT-4o. The rule of the game is simple: you need to score 50 points to beat the computer, which can be quite challenging.

Beating the computer can be a little tricky due to its accurate ball-tracking AI. However, there's a way to win if you decide to open your console and cheat. Unfortunately, the game is not supported on mobile devices or tablets.
